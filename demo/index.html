JavaScript API

generateAvatar(options)

seedstringAny string — same seed always produces the same avatar
stylestring?marble | beam | bauhaus | pixel | ring | glitch | constellation | emoticon | nebula | wireframe | halftone | isometric | spectrum
moodstring?none | happy | sad | angry | calm | chaotic
animstring?none | float | morph | glimmer
sizenumber?Pixel dimension (default: 40)
colorsstring[]?5 hex strings to override palette
radiusnumber?Border radius fraction 0–0.5 (default: 0.22)

import { generateAvatar } from 'void-avatars'; const svg = generateAvatar({ seed: 'user-42', style: 'constellation', mood: 'calm', anim: 'float', size: 80, });
